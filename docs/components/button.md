# Button

## Import

```ts
import '@aesc/components/button';
```

## Ejemplos

```html
<ae-button>Primary</ae-button>
<ae-button variant="secondary">Secondary</ae-button>
<ae-button variant="ghost">Ghost</ae-button>
<ae-button disabled>Disabled</ae-button>
```

## API

- `variant`: `primary | secondary | ghost`
- `size`: `sm | md | lg`
- `disabled`: boolean
